# GroupMe Event Info Bot

A Python-based GroupMe bot that automatically posts event information to a group chat.

## Features
- Fetches and formats upcoming event data
- Automatically posts event details to the group
- Designed to reduce manual coordination for student organizations

## Technologies Used
- Python
- GroupMe API
- REST APIs
- JSON parsing

## How to Run
1. Clone the repository
   git clone https://github.com/yourusername/GroupMeBot.git
2. Install dependencies
   pip install -r requirements.txt
3. Set environment variables for the GroupMe bot token
4. Run the bot
   python nu_chi_event_info_bot.py
